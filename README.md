🔐 Secure Message Encryption System

A cybersecurity-focused secure messaging application built using Python, Streamlit, Fernet AES Encryption, and QR Code Technology for encrypted communication and secure QR-based message sharing.

🚀 Features
Secure Message Encryption & Decryption
Password-Protected Decryption
Fernet AES Symmetric Encryption
QR Code Generation for Encrypted Messages
QR Code Upload & Auto Decryption
Secret Key Generation & Management
Downloadable Encrypted QR Codes
Interactive Streamlit Web Interface

Streamlit
Cryptography (Fernet AES)

🔒 How It Works
Generate a secure encryption key
Encrypt confidential messages using Fernet AES
Convert encrypted text into QR codes
Share QR securely with users
Upload/scan QR code for automatic decryption
Verify password for protected access
